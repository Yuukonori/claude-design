Flat bordered container for grouping content; no shadow, sharp corners.

```jsx
<Card header={<strong>Inspector</strong>}>Properties go here</Card>
<Card interactive padding="sm">Clickable row</Card>
```

Props: `padding` (`none`/`sm`/`md`/`lg`), `interactive` (hover border), `header`, `footer`. Inline cards never use shadow — only borders.
